# proxy.py
A simple http proxy server

# Usage
```
usage: proxy.py [-h] [--ipv4] [--log {DEBUG,INFO,WARNING}] [address] [port]

HTTP proxy server

positional arguments:
  address
  port

optional arguments:
  -h, --help            show this help message and exit
  --ipv4                IPv4 only
  --log {DEBUG,INFO,WARNING}
```
